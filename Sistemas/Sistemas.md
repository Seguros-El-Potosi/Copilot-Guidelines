
## Servicios

Seguros El Potosí cuenta con los siguientes servicios, los cuales están disponibles para ser implementados o utilizados en los sistemas desarrollados:

1. **Azure Storage**: API para subir y descargar archivos de cualquier tipo a la nube (por ejemplo: PDF, PNG, JPG, entre otros).
2. **SMS API**: API para el envío de mensajes de texto (SMS).
